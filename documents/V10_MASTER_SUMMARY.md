## 🔧 How It Works

### Header Image
1. Set in meta tag: `<meta name="article:header-image" content="URL"/>`
2. JavaScript auto-loads on page load
3. Applies gradient overlay
4. Sets to cover/center

### Navigation Cards
1. Edit HTML directly (lines 1895-1913)
2. Update `href` attribute with full URL
3. Update `nav-card-title` text
4. No JavaScript needed

### Social Sharing
1. Click share button
2. JavaScript opens share window
3. Auto-fills current URL and title
4. Works with Facebook, Twitter, LinkedIn, Email

### Table of Contents
1. Scroll triggers active section check
2. JavaScript highlights current section
3. Smooth scroll on click
4. Auto-updates as you scroll

## 🆘 Quick Troubleshooting

| Problem | Solution |
|---------|----------|
| Header image not loading | Check meta tag URL, verify it works in browser |
| Navigation broken | Use full URLs, not relative paths |
| Mobile menu not working | Check JavaScript console for errors |
| Styling looks wrong | Clear browser cache, hard refresh |
| Links not working | Verify GitHub Pages has built (wait 2 min) |
